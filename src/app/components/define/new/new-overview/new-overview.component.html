Attribute Range rule
This type of rule compares a column to a numeric range.
Attribute Compare rule
This type of rule compares a column and a value.
Attribute List rule
This type of rule Compares a column against a list of values.
Attribute Other rule
This type of rule Compares a value to a self written condition.
Tuple Compare rule
This type of rule compares between two columns.
Tuple Other rule
This type of rule uses 2 columns to compare.
Inter-Entity Compare rule
This type of rule compares columns from two different tables.
Entity Other rule
This type of rule uses 2 columns from different tables to compare.
Modify rule
This rule prevents action from happening if condition is true.
